# תיקיית קבצי עיבוד

תיקייה זו מכילה קבצי ביניים שנוצרים מתוך דוחות ה-PDF לפני בניית מאגר הווקטורים.

כל תת-תיקייה מייצגת אוסף דוחות אחר, לפי מזהה מסוג:

```text
reports_<hash>
```

בכל אוסף נשמרים הקבצים הבאים:

```text
pages.jsonl
chunks.jsonl
metadata.csv
manifest.json
```

`pages.jsonl` שומר טקסט שחולץ לפי עמוד.

`chunks.jsonl` שומר את קטעי הטקסט שמוכנסים למאגר ה-RAG.

`metadata.csv` שומר טבלת בקרה עם מקור, עמוד ומזהה chunk.

`manifest.json` שומר את הגדרות הבנייה: שמות הדוחות, מודל ה-Embeddings, גודל chunk ומספר הרשומות.
